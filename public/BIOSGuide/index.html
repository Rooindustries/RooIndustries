Is Your PC Underperforming?

Here's a shocking truth: Your RAM is probably running at half its rated speed right now. And your CPU? It might be holding back too. This isn't your fault—it's how computers ship from the factory.

The Hidden Performance Problem

When you buy DDR4-3600 or DDR5-6000 RAM, that speed on the box? Your PC doesn't use it by default. Instead, it runs at the "safe" base speed:

The Factory Settings Problem

Manufacturers set everything to the lowest common denominator for compatibility. Great for avoiding returns. Terrible for your gaming performance. You paid for speed you're not getting.

How to Test Your Current Performance

Before we optimize anything, let's see where you stand right now. This gives you a "before" score to compare against your "after" results.

Step 1: Check Your RAM Speed Right Now

  1. Press Ctrl + Shift + Esc to open Task Manager
  2. Click the "Performance" tab at the top
  3. Click "Memory" on the left sidebar
  4. Look at the "Speed" value in the top right
What Your Speed Means

If you see 2133 MHz (DDR4) or 4800 MHz (DDR5): Your RAM is running at base speed. XMP/EXPO is NOT enabled. You're leaving significant performance on the table.

If you see your rated speed (3200/3600/6000 MHz etc.): XMP/EXPO is already enabled. Skip to Part 6 for CPU optimization.

Step 2: Run Free Benchmark Tests

What is BIOS & Why It Matters

BIOS is your computer's control center that runs BEFORE Windows even starts. It controls how your hardware behaves at the deepest level.

BIOS in Plain English

Think of BIOS like this:

Analogy What It Means
Car engine settings BIOS controls how "fast" your hardware is allowed to run
Home circuit breaker BIOS controls power delivery to your components
Phone's hidden settings menu BIOS has settings you can't access from Windows

What BIOS Controls

RAM Speed

How fast your memory runs. Default is slow. XMP/EXPO unlocks full speed.

CPU Power

How much power your CPU gets. PBO unlocks higher clocks on AMD.

Boot Order

Which drive Windows loads from. Important for SSDs.

Fan Curves

How fast your cooling fans spin at different temperatures.

UEFI vs BIOS: What's the Difference?

UEFI is just the modern, graphical BIOS. For this guide, they mean the same thing. All modern boards use UEFI.

Why Manufacturers Ship With "Safe" Settings

Motherboard makers want zero returns. So they set everything conservatively—your RAM runs at base speed instead of rated. Your CPU stays within safe limits even if it could boost higher. These settings guarantee compatibility but sacrifice performance.

Is This Safe?

Yes. XMP/EXPO are factory-tested profiles. PBO is AMD's official performance feature. The worst that can happen is a reboot and an auto-revert to safe settings.

How to Enter BIOS (All Brands)

To change any BIOS settings, you need to enter BIOS during startup. Here's exactly how to do it for every major motherboard brand.

Quick Reference: BIOS Entry Keys

ASUS
DEL or F2
MSI
DEL
Gigabyte
DEL
ASRock
DEL or F2

Step-by-Step Instructions

  1. Restart your computer — Click Start → Power → Restart
  2. Spam the BIOS key immediately — As soon as the screen goes black, start pressing DEL repeatedly
  3. Keep pressing until BIOS appears — You'll see a graphical interface with your motherboard's logo
  4. If Windows loads instead — You missed the window. Restart and try again

Using a Wireless Keyboard?

Some wireless keyboards don't work during boot. If you can't enter BIOS, try a wired USB keyboard temporarily.

Alternative Method: Enter BIOS from Windows

If the key method doesn't work (Fast Boot enabled), use this method:

  1. Open Settings → System → Recovery
  2. Under "Advanced startup," click "Restart now"
  3. After restart, click Troubleshoot → Advanced options
  4. Click "UEFI Firmware Settings" → Restart
  5. Your PC will restart directly into BIOS

Understanding EZ Mode vs Advanced Mode

Mode What It Shows When to Use
EZ Mode Simplified view with basic settings, XMP toggle often visible Quick XMP enable, fan curves, system info
Advanced Mode Full settings tree with all options PBO settings, detailed RAM timings, power limits
How to Switch Between Modes

ASUS/MSI/Gigabyte: Press F7
ASRock: Press F6

XMP & EXPO — Unlock Your RAM Speed

This is the single most impactful BIOS change for gaming performance. One click can give you 15-30% more FPS in CPU-limited games.

What is XMP / EXPO / DOCP?

Name Platform What It Does
XMP
Extreme Memory Profile 		Intel Factory-tested profile stored on your RAM. Enables rated speed with one click.
EXPO
Extended Profiles for Overclocking 		AMD (AM5) AMD's version of XMP for DDR5 memory. Optimized for Ryzen 7000/9000.
DOCP / A-XMP AMD (AM4) AMD-compatible versions of XMP for older platforms. Same result.
Which One Do You Need?

Intel CPU? Look for XMP
AMD Ryzen 7000/9000 (AM5)? Look for EXPO first, then XMP if not available
AMD Ryzen 3000/5000 (AM4)? Look for DOCP, A-XMP, or just XMP

XMP/EXPO is NOT Risky Overclocking

These profiles are factory-tested. You're just enabling the speed you paid for. If it doesn't work, your PC usually auto-recovers after a few failed boots.

ASUS
ASUS Motherboards

Method 1: EZ Mode (Fastest)

  1. Enter BIOS by pressing DEL during startup
  2. You'll land in EZ Mode by default
  3. Find "X.M.P." or "EXPO" on the main screen
  4. Select "Enabled" or "Profile 1"
  5. Press F10 to save and exit

Method 2: Advanced Mode

AI Tweaker → Ai Overclock Tuner → XMP I / XMP II / EXPO I

  1. Press F7 to enter Advanced Mode
  2. Go to "AI Tweaker" (or "Extreme Tweaker")
  3. Change "Ai Overclock Tuner" from Auto to XMP I, XMP II, or EXPO
  4. Press F10 → Save and Exit
XMP I vs XMP II — Which to Choose?

XMP I: ASUS optimized settings. Try first.
XMP II: Uses the stick’s raw profile. Use if XMP I causes issues.

2. Update BIOS
3. Try a lower speed (e.g., 5600 instead of 6000)
4. Ensure RAM is in A2 + B2 (slots 2 and 4)

When XMP/EXPO Doesn't Work

XMP/EXPO failing is common. Don’t panic—there are two main causes, and both have fixes.

Why XMP/EXPO Fails

Cause What's Happening Signs
Memory Controller Limit Your CPU's memory controller can't handle the frequency. No POST, boot loops, or crashes under load
Voltage Issue RAM voltage isn’t being applied correctly (common on picky combos). Random crashes, memtest errors, instability

Diagnostic Steps

PC Won't POST After Enabling XMP

Wait 30–60 seconds (memory training can take time). If nothing happens, hold power for 10 seconds. Many boards auto-recover after a few failed boots. If not, clear CMOS (Part 8).

XMP Boots But System Is Unstable

In BIOS, set DRAM Voltage manually: DDR4: 1.35V | DDR5: 1.35–1.40V. For DDR5 also match VDDQ to VDD.

XMP Works at Lower Speed But Not Rated Speed

Your CPU/board combo may not handle the rated frequency. Drop 200–400 MHz, or use Profile 2. This is common and not a defect.

Voltage Quick Reference

Platform Voltage for Frequency Issues Voltage for Timing Issues
Intel SA (System Agent) — try 1.25–1.35V DRAM Voltage — match XMP spec
AMD SoC Voltage — try 1.10–1.20V DRAM VDD/VDDQ — match EXPO spec

Dual-Rank RAM Warning

Dual-rank kits often need more voltage and may not hit the same XMP speed. Dropping 200–400 MHz is a normal, practical solution.

RAM Slot Configuration

Wrong slots = worse performance or XMP failure. This trips up more people than you'd expect.

Correct Slot Placement

For a 2-stick kit on a 4-slot motherboard, use slots 2 and 4 (A2 + B2):

[CPU SOCKET THIS SIDE]

Slot 1 EMPTY
Slot 2 RAM
Slot 3 EMPTY
Slot 4 RAM

Slots 2 and 4 enable dual-channel and have better signal integrity on most boards.

Why Not Slots 1 and 3?

Slots 2 and 4 typically have better signal integrity. Wrong slots can prevent XMP from working even if everything else is correct.

4-Stick Configurations

If you have 4 sticks, fill all slots. But know 4 sticks is harder on the memory controller; you may need to lower speed.

Brand-Specific Issues

Each motherboard brand has quirks. Knowing them saves hours.

ASUS

Generally best memory training. Try XMP I first. If unstable, use XMP II or update BIOS.

MSI

Some boards undervolt DRAM on Auto—set DRAM voltage manually to spec if unstable.

Gigabyte

Look under Tweaker for XMP. If boot loops, wait longer for memory training on first boot.

ASRock

Sometimes hides XMP deeper under OC Tweaker. Ensure you load the profile, not just set frequency.

Pro Tip

If you updated BIOS and XMP still fails, the fastest "real-world" fix is often: drop frequency 200–400 MHz, keep voltage at spec, and enjoy stable performance.

PBO — AMD CPU Performance Boost

If you have an AMD Ryzen CPU, PBO (Precision Boost Overdrive) can increase boost clocks automatically, improving FPS and 1% lows in CPU-limited games.

What PBO Actually Does

Setting What It Changes Result
PBO: Enabled Raises power/current limits within safe rules Higher sustained boost clocks
Curve Optimizer Undervolts per-core intelligently More boost + lower temps (best tweak)
Too aggressive CO Undervolt too much Crashes / WHEA errors

Intel Users

PBO is AMD-only. If you have Intel, skip to Part 7. Your main win was XMP.

Quick PBO Setup (Safe Defaults)

  1. Enter BIOS → Advanced Mode (F7 on most boards)
  2. Find PBO / Precision Boost Overdrive
  3. Set PBO to Enabled (or Advanced)
  4. Leave limits on Auto for now
  5. Save & Exit (F10)

Expected Result

Most Ryzen CPUs will boost a bit higher under load, especially with good cooling. Real gains show up in CPU-heavy games as better 1% lows and less stutter.

Other Essential BIOS Settings

These aren’t always performance boosters, but they improve stability, compatibility, and quality-of-life.

Resize BAR (ReBAR)

Enables CPU to access more GPU memory at once. Good for many modern games. Usually: Enabled.

Above 4G Decoding

Often required for ReBAR. Enable it if you enable ReBAR.

Fan Curves

Set a sane curve: quiet at idle, ramps up under load. Helps temps = better boost clocks.

Boot Order

Make sure your OS drive is first. Fixes random “no boot device” issues after BIOS resets.

Typical Paths

Advanced → PCIe / PCI Subsystem → Re-Size BAR Support

Advanced → PCIe / PCI Subsystem → Above 4G Decoding

If You Don’t See ReBAR

Update BIOS and GPU drivers. Some older GPUs/boards don’t support it. If unsupported, don’t worry—XMP/PBO were the big wins.

Safety & Recovery

If something goes wrong (boot loop after XMP/PBO), your PC is not “dead.” Use these recovery steps.

1) Auto-Recovery (Most Common)

  1. After enabling XMP, first boot can take 30–60 seconds (memory training)
  2. If it fails, many boards will try 2–3 times, then revert to safe defaults
  3. If you see BIOS again with a message like “settings were reset,” just re-enable XMP with a lower profile

2) Clear CMOS (Hard Reset)

Before You Do This

Clearing CMOS resets BIOS settings (XMP/PBO/boot order). It does NOT delete Windows or your files.

  1. Power off the PC and switch PSU off (or unplug)
  2. Press the power button once (drains leftover power)
  3. Use one of these methods:
    • Press the motherboard Clear CMOS button (if you have one)
    • Short the CLR_CMOS pins with a screwdriver for 5–10 seconds (check your manual)
    • Remove the coin-cell battery for 2–3 minutes, then reinstall
  4. Boot up → enter BIOS → set correct boot drive → re-enable XMP (try Profile 2 or lower speed)

Fastest “Back to Stable” Plan

If XMP fails: clear CMOS → enable XMP → set DRAM voltage to spec → lower speed by 200 MHz. You’ll get most of the gains with maximum stability.
