# 3D model attribution

The homepage hero GPU model is based on "NVIDIA RTX 5090 Founders Edition" by DavidWorks
(https://sketchfab.com/3d-models/nvidia-rtx-5090-founders-edition-5934a7ddfa144951ac66eb8cab13f220),
licensed under CC-BY-4.0 (https://creativecommons.org/licenses/by/4.0/).
Modifications: remapped into six named parts for the scroll teardown, internals replaced,
textures re-embedded as WebP, scale/orientation adjusted.

A user-facing credit line should accompany the model per CC-BY-4.0. Planned placement:
site footer credits (pending operator's placement choice).
